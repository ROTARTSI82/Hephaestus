◆ init_logging()

void hp::init_logging ( bool  use_single_file = true,
bool  use_unique_file = false 
)

Initialize Hephaestus logging (which uses SPDLOG).

Call this function before logging anything, as any logs before a call to this function is lost. This function initializes SPDLOG, so see https://github.com/gabime/spdlog.

Parameters
use_single_fileIf true, latest.log would be created and logged to.
use_unique_fileIf true, a logfile with the current datetime as a name under directory logs would be created and logged to. If the directory logs doesn't exist, it would be treated as if this parameter was set to false.

◆ init_threads()

void hp::init_threads ( unsigned  num_threads = 0)

Initialize/Construct the default thread pool.

Best practice is to call init_threads at program startup and quit_threads at exit.

Parameters
num_threadsThe number of threads to assign to the thread pool. If set to 0, a reasonable number of threads would be automatically selected depending on hardware. If this automatic selection fails, this value would default to 8.

◆ io_service

boost::asio::io_service * hp::io_service

Interface for submitting asynchronous tasks.

Once the default thread pool has been constructed, you can submit a task using "io_service->post". For example, "io_service->post(boost::bind(printf, "Test"));" would queue "printf("Test")" to be called by the threadpool. See boost documentation for more details.

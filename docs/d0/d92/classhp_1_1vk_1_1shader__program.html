Detailed Description

An abstraction of graphics pipelines (aka vk::Pipeline objects). See hp::vk::window::new_shader_program.

shader_programs use the buffer_layouts that are bound at the time of construction. If no buffer_layout is bound, the default layout is used. (If the default layout has not yet been constructed, it would automatically be built with build_default_layout()). The buffer_layout used by a shader_program can be changed, but requires the pipeline to be rebuilt (There is no dynamic state for vertex input states present in Vulkan). To change the layout used by a shader program, simply the modify buffer_layout::bound_lyos, call buffer_layout::rebuild_bound_info(), and call hp::vk::shader_program::rebuild_pipeline().

Detailed Description

Contains vertex data in default buffer_layout format.

This struct contains vertex data in the format specified in the default buffer_layout. Therefore, simply writing an array of this struct to a vertex_buffer would be valid if the default buffer_layout was bound at the creation of the active shader_program. The format of vertex is the following: location=0 is a vec2 representing position, and location=1 is a vec3 representing color.

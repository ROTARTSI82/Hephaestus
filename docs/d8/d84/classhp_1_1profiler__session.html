Detailed Description

Profiler sessions represent files that reports are written to.

You can utilize discrete profiler sessions to output different profiles to different files.

◆ close()

void hp::profiler_session::close ( )

Closes the file.

Note
This function does not finish writing queued reports, so reports in queue WILL NOT be written! Best practice is to call hp::profiler_session::flush_all() before calling this function.

◆ flush_all()

void hp::profiler_session::flush_all ( )

Writes all queued profile reports to the file.

Note
You MUST call this function before calling close(), or reports in queue will NOT be written!

◆ default_session

static profiler_session * hp::profiler_session::default_session
static

A user-defined pointer to the session to be used by HP_START_PROFILER

The user should assign this variable before using HP_START_PROFILER. Example Usage: hp::profiler_session::default_session = new profiler_session("Profiling Session"); would set the default profiler session for use in HP_START_PROFILER

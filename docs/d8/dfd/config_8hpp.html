◆ HP_VK_VALIDATION_LAYERS_ENABLED

#define HP_VK_VALIDATION_LAYERS_ENABLED

Global flag to enable/disable Vulkan validation layers. See vk/window.hpp.

Warning
Disabling would entirely disable vulkan error checking, so any vulkan errors would just segfault. Save yourself the pain of spending hours of debugging and enable this feature while debugging. Once your program is free of errors, disable this in release to get rid of the overhead.

Enabled by default. (Should be disabled in release, rebuild required.)
